# Como executar o projeto

```bash
# clonar repositório
git clone git@github.com:vitouz/Admin-Dashboard.git

# entrar na pasta do projeto back end
cd admForm

## Front end 
Pré-requisitos: npm / yarn

# instalar dependências
yarn install

# executar o projeto
yarn dev
```
